Thuật toán Xác định Khoảng Đơn điệu và Khoảng Cách lập của Hàm số
Input:
	Mảng x[] chứa các mốc dữ liệu
	Mảng y[] chứa giá trị tương ứng
Output:
	Mảng mono[] chứa các khoảng đơn điệu [x_start, x_end, kiểu] (kiểu: 1=tăng, -1=giảm)
	Mảng isol[] chứa các khoảng cách lập [x_start, x_end]

Bước 1: Check điều kiện Input
	Kiểm tra độ dài x[] và y[] có bằng nhau không, nếu không dừng
	Kiểm tra len(x) > 1, nếu không dừng

Bước 2: Khởi tạo các biến, mảng trung gian
	n = len(x) - 1
	mono[] = mảng rỗng		// lưu các khoảng đơn điệu
	isol[] = mảng rỗng		// lưu các khoảng cách lập
	trend = 0				// xu hướng hiện tại: 1=tăng, -1=giảm, 0=chưa xác định
	x_start = x[0]			// bắt đầu khoảng đơn điệu
	eps = 1e-6				// sai số nhỏ để so sánh

Bước 3: Xác định các khoảng đơn điệu dựa trên xu hướng tăng/giảm của y
	For i = 0 to n-1:
		If abs(y[i+1] - y[i]) < eps:
			// y[i] ≈ y[i+1], tiếp tục xu hướng hiện tại
			continue
		Else If y[i+1] > y[i]:
			// y tăng
			If trend == -1:
				// Đổi từ giảm sang tăng, lưu khoảng giảm
				Thêm [x_start, x[i], -1] vào mono
				x_start = x[i]
			trend = 1
		Else:
			// y giảm
			If trend == 1:
				// Đổi từ tăng sang giảm, lưu khoảng tăng
				Thêm [x_start, x[i], 1] vào mono
				x_start = x[i]
			trend = -1
	// Lưu khoảng cuối cùng
	If trend ≠ 0:
		Thêm [x_start, x[n], trend] vào mono

Bước 4: Xác định các khoảng cách lập dựa trên đổi dấu của y
	For i = 0 to n-1:
		If y[i] * y[i+1] < 0:
			// f(x[i]) và f(x[i+1]) trái dấu, chắc chắn có nghiệm trong (x[i], x[i+1])
			Thêm [x[i], x[i+1]] vào isol
		Else If abs(y[i]) < eps:
			// y[i] ≈ 0, gần bằng 0
			If i == 0 Or i == n:
				// Ở đầu hoặc cuối, lưu khoảng với điểm tiếp theo
				If i == 0 And y[1] ≠ 0:
					Thêm [x[0], x[1]] vào isol
				If i == n And y[n-1] ≠ 0:
					Thêm [x[n-1], x[n]] vào isol
			Else:
				// Ở giữa, kiểm tra đổi xu hướng
				If (y[i-1] > 0 And y[i+1] < 0) Or (y[i-1] < 0 And y[i+1] > 0):
					Thêm [x[i-1], x[i+1]] vào isol

Bước 5: Return kết quả
	Return [mono, isol]
